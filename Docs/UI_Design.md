# 사후르 키우기 화면 구성 (UI Design)

## 1. 기본 가이드 (Basic Guide)
### 해상도 및 레이아웃
- **Reference Resolution**: 1080 x 2160
- **대응 전략**:
  - 9:18 비율보다 세로가 짧은 경우: **레터 박스(Letter Box)** 처리
  - 9:18 비율보다 세로가 긴 경우: 상단바와 하단바가 아닌 **중앙 영역(Stretch)**이 늘어나서 대응
- **상단 바 (Top Bar)**:
  - 현재 화면에서 필요한 재화만 표시
  - 옵션 및 배속 옵션 버튼 위치
- **하단 바 (Bottom Bar)**:
  - 주요 메뉴 탭 구성: **강화 / 스킬 / 장비 / 퀴즈**
  - 확장 가능성을 고려하여 설계 (Scrollable 또는 탭 추가 용이 구조)
  - 연출 참고: '자라나라 머리머리' 스타일

## 2. 메인 전투 화면 (Main Battle Screen)

- **로봇 (캐릭터)**: 머리 위에 현재 레벨을 빌보드(Billboard) 형태로 표시
- **상단**: 가이드 퀘스트 표시
- **양쪽 사이드**: 출석부 등 사이드 콘텐츠 버튼 배치
- **업그레이드 패널**:
  - 기본적으로 열려 있으며, 특정 레벨 도달 시 잠금 해제되는 방식 (자물쇠 아이콘 → 오픈)

## 3. 스킬 화면 (Skill Screen)
- **구성**: 팝업(Popup) 형태
- **스킬 종류**: 액티브(Active) / 패시브(Passive) 각 1개씩 기본 구성
- **해금 조건**:
  - 1-10 / 3-10 / 5-10 / 7-10 스테이지 클리어 시 각 스킬 해금
- **슬롯 스펙**:
  - **NEW 표시**: 처음 획득 시 표시
  - **장착 토글**: 빨간색 'E' (장착됨) ↔ 회색 'E' (해제됨) 터치로 토글
  - **상세 정보**: 슬롯 클릭 시 팝업

## 4. 장비 화면 (Equipment Screen)
- **구성**: 팝업(Popup) 형태
- **슬롯**: 총 6개 (현재는 무기 하나지만 6개 짝을 맞추는 구조로 설계)
- **강화**:
  - 재화: **에메랄드**
  - 효과: 강화 시 '장착 시 공격력'과 '보유 효과'가 동시에 증가
- **기능**: 장비 뽑기 / 장비 강화 버튼

## 5. 퀴즈 화면 (Quiz Screen)
- **컨셉**: '저질 점프맵' 영상 + '스폰지밥 퀴즈' 스타일의 오버레이 퀴즈
- **난이도**: Easy / Normal / Hard / Super Hard / Hell
  - 난이도에 따른 보상 차이는 크지 않음 (성취감 목적)
- **입장 조건**: 티켓 필요
- **기능**:
  - **리워드 광고**: 시청 시 최고 기록으로 즉시 완료 가능
  - **부활**: 퀴즈 실패 시 인터스티셜 광고 시청 후 부활 가능 (타이머 제공)
- **연출**: 자막/TTS 등을 활용하여 '쓰레기 브레인롯 릴스' 감성 구현
